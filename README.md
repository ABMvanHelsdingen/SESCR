# SESCR
Code and Data for A Hawkes-type Model extending Spatial Capture-Recapture, Biometrics, 2024*

## Simulations
Sim-SESCR.R and Sim-OU.R simulate data for Sections 3.1 and 3.2. Sim-SCR.R simulates data for Appendix B. 
SimulateFunctions.R contains functions used by all scripts. 
The data is saved as .csv files for further use. 

## Fitting
MLE-SESCR.R and MLE-OU.R fit our SESCR model by MLE as described in Section 2. 
The data must be in the format generated by the two simulation files above. 
MLEFunction.cpp is the TMB template, which is compiled if necessary by the two fitting files.
FitFunctions.R functions for processing and preparing data and the mask. 


Bayesian-SCR.R fits data from Sim-SCR.R in a Bayesian framework.
NIMBLE functions are contained within FitFunctions.R
